Boolean Identities and Laws

The symbols used are + for OR, · for AND and a bar for NOT. Other exam boards may use V for OR, ^ for AND and ¬ for NOT

The order of precedence in Boolean Algebra is Brackets > AND > OR

You can sort of treat AND operations as multiplication and OR operations as addition

Name Identity
Identity Law A + 0 = A
A · 1 = A
Identity (Domination) Law A + 1 = 1
A · 0 = 0
Idempotent Law A + A = A
A · A = A
Complement Law A + A = 1
A · A = 0
Double Negation Law A = A
Commutative Law A + B = B + A
A · B = B · A
Associative Law (A + B) + C = A + (B + C)
(A · B) · C = A · (B · C)
Distributive Law A · (B + C) = (A · B) + (A · C)
A + (B · C) = (A + B) · (A + C)
Absorption Law A + (A · B) = A
A · (A + B) = A
De Morgan’s Theorems (A · B) = A + B
(A + B) = A · B
Redundancy Laws (Consensus Theorem) A + (A · B) = A + B
A · (A + B) = A · B
